# [2026-1] 레포트 - 17기_한규리, 17기_황지영

## GARCH 모형을 활용한 인도네시아 루피아(IDR) 환리스크 분석 및 동적 헤징 전략

## 레포트 소개
* 최근 인도네시아는 디지털 립프로깅과 대규모 인프라 투자가 집중되며 높은 기대 수익을 제공하는 주요 투자처로 자리 잡았습니다. 그러나 과거 테이퍼 탠트럼 사례에서 확인되듯, 루피아화(IDR)의 높은 환율 변동성은 현지 진출 기업의 실질적인 수익성을 훼손하는 리스크 요인으로 작용합니다. 본 레포트에서는 인도네시아 진출 시 수반되는 환리스크를 GARCH 모형을 적용하여 정량적으로 계량화하였습니다. 나아가 CDS 프리미엄을 바탕으로 시장 국면에 따라 통화스왑 등 어떤 헤징 전략을 선택하는 것이 비용 효율적인지 실무적인 가이드라인을 제시하였습니다.

## 레포트 요약
* 인도네시아 진출 기업의 환리스크 관리를 위해 IDR/KRW 환율 변동성을 GARCH(1,1) 모형으로 정량화하여 분석했습니다. 분석 결과 루피아화는 대외 충격 발생 시 변동성 급등이 장기간 지속되는 뚜렷한 특징을 보였으며, 1일 최대 예상 손실(VaR)은 약 1.07% 수준으로 산출되었습니다. 또한 통화스왑(CRS) 헤징 비용은 단순 내외금리차를 넘어 국가 신용 위험을 나타내는 CDS 프리미엄에 크게 좌우됨을 확인했습니다. 결론적으로 현재 인도네시아의 CDS 스프레드가 역사적 평균을 밑돌고 있으므로 현시점에서 선제적으로 통화스왑을 체결해 환노출 비용을 확정 짓는 것이 가장 비용 효율적인 전략이라는 결론을 도출할 수 있었습니다.
