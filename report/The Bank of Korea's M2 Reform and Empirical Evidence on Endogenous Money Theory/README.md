# 한국은행의 M2 편제 개편과 내생적 화폐 이론에 대한 실증 분석

* **작성:** 17기 박세헌, 18기 강지호

## 레포트 소개
* 본 연구는 한국은행의 최근 M2 개편 사례(수익증권 제외)를 바탕으로, 시중 통화량 급증이 자산시장발 포트폴리오 이동에 의한 것임을 VAR 모형과 그레인저 인과분석, 충격반응함수 통해 계량적으로 검증합니다.

## 레포트 요약
* 본 연구는 2025년 한국은행의 M2 개편 타당성을 실증 분석하여, 금융불안 발생 시 신 M2가 구 M2와 달리 즉각적이고 강력한 팽창 반응을 보임을 입증했습니다. 이는 위기 시 안전자산 선호에 따른 '포트폴리오 내생성'을 명확히 반영한 결과를 확인하였습니다.

## 파일 설명
* `report_code.ipynb`: VAR모형 설정과 충격반응함수 코드
